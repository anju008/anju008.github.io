# Anju Shaji – Marketing Portfolio

This repository contains a modern, responsive portfolio for Anju Shaji. The site showcases selected work, background, and contact information using lightweight HTML and Tailwind CSS.

## Structure
- `index.html` – home page with featured projects
- `about.html` – background, skills, and experience
- `contact.html` – contact details and FAQs
- `loro-piana.html` – case study
- `vinted.html` – case study

## Development
The site uses CDN-hosted [Tailwind CSS](https://tailwindcss.com) for styling. No build step is required; open any HTML file in a browser to view.
